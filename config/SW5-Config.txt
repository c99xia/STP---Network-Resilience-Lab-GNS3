Switch 5 Configuration

Initially Configuration

enable
configure terminal
hostname SW5
line console 0
logg synchronous
exec-timeout 0

1. Configure PC2 in VLAN 10

int e0/0
switchport mode access
switchport access vlan 10

3. Order of STP priorities: SW5 - SW1 - SW3 - SW7 - SW2 - SW4 - SW6

configure terminal
spanning-tree vlan 1,10,20,30,99 priority 8192 


SW1.e1/3 - SW5.e1/1 is backup

int e1/3 
spanning-tree port-priority 64


SW5 - SW3 is a port channel (assume bandwidth 12 Mbps)

int range e2/0 - 1
channel-group 1 mode active
int po1 
bandwidth 12000

5. Route VLAN 20 to SW1

int range po1 , e1/1 , e1/3
switchport trunk encapsulation dot1q
switchport mode trunk
switchport trunk native vlan 99
