Switch 7 Configuration

Initially Configuration

enable
configure terminal
hostname SW7
line console 0
logg synchronous
exec-timeout 0

3. Order of STP priorities: SW5 - SW1 - SW3 - SW7 - SW2 - SW4 - SW6

conf t
spanning-tree vlan 1,10,20,30,99 priority 32768 



5. Route VLAN 20 to SW1

int e0/3
switchport trunk encapsulation dot1q
switchport mode trunk
switchport trunk native vlan 99



7. Install routed link in SW7 - R1

conf t
int e1/1
no switchport 
ip add 192.168.98.2 255.255.255.0
no shut
ip routing
router rip
ver 2
net 0.0.0.0
end


8. Route VLAN 99 to SW7.

conf t
int vlan 99
ip add 192.168.99.7 255.255.255.0 
no shut
